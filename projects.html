An easy to install, fast to get started, custom boilerplate.

I built this for myself. It makes my life a little bit easier. Instead of fumbling through third-party frameworks and unknown css files I can get a project started with three commands. 

                    
mkdir newProject
git clone https://github.com/mattcass/boilerplate.git
npm install
