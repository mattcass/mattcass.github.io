European Performance Engineering

EPE services porsche autmobiles. From high end track specific cars to daily drivers. The site was designed and developed with a clean minimal aesthetic to display the varying services and products offered.

The new web interface presents the brand in a sophisticated light and takes advantage of modern technologies allowing customers to get location and contact information directly from their mobile phones.

Janky Donuts

Built in association with the !ND!V!DUALS COLLECTIVE, a Boston based art group that specializes in repurposing reclaimed wood through interior design, gallery work, and sculptures.

Because of the small size of the project the site was built without a framework. This allowed full control of the UI elements and presented a welcome challenge creating a lean css file structure that still worked cross browser. Lacking a framework to fall back onto, media queries were utilized on a few key elements to keep the site responsive.

www.jankydonuts.com

Winston Macdonald

Winston Macdonald, a Boston based videographer needed a clean, content driven website that would allow him to quickly display his work to prospective clients in a professional manner.

The site was built around the Skeleton grid framework to ensure a responsive design. With a clear focus on contact information as well as his most recent work, visitors do not have to spend extra time searching for what they need.

www.winstonmacdonald.com
